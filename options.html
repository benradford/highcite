How it works

If the citation contains a DOI — HighCite queries the CrossRef REST API (free, no account needed) and builds a precise BibTeX entry from the publisher metadata.

Otherwise — it parses the text locally using format-aware rules for APA, MLA, Chicago, Vancouver, and IEEE citations.

CrossRef polite pool (optional)

CrossRef gives extensions that identify themselves preferential rate limits. If set, your email is sent as a query parameter to api.crossref.org only — nowhere else.
