HighCite Settings
No API key required.

How it works

1. DOI detected — queries the CrossRef REST API (free, no account needed) for precise publisher metadata.

2. Gemini Nano — Chrome's built-in on-device AI model parses the citation text. Can be disabled below.

3. Regex fallback — format-aware rules for APA, MLA, Chicago, Vancouver, and IEEE.

Gemini Nano (optional)

Use on-device AI for parsing
When enabled, HighCite tries Chrome's built-in Gemini Nano model before falling back to regex rules. Requires Chrome 138+ with the model downloaded.
🔒 All processing is local. Citation text is never sent to Google, Anthropic, or any other cloud provider.

CrossRef polite pool (optional)

CrossRef gives extensions that identify themselves preferential rate limits. If set, your email is sent as a query parameter to api.crossref.org only — nowhere else.
