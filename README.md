HighCite
==================

- Benjamin J. Radford  
- Claude Sonnet 4.6

HighCite is a Chrome browser extension for creating paste-able **BibTeX** entries. 

![Demonstration of HighCite extension.](images/demo.png "Demonstration of HighCite extension.")

## Usage

1. Highlight the academic citation on the website. Include highlighting the DOI if available. Works best for APA / MLA / Chicago / Vancouver / IEEE formats.
2. Right click the highlighted citation and select **Convert to BibTex**.
3. Paste directly into your .bib file. 

## Installation

1. Clone this repository to your local computer.
2. In Chrome's address bar, navigate to: `chrome://extensions`
3. In the upper right corner, turn on **Developer mode**.
4. Click **Load unpacked**.
5. Navigate to this repository on your computer.
6. Click **Select**.
